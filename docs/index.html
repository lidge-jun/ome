OME

Orchestrated Multi-agent Engine — Spawn and orchestrate AI agent CLIs as employees from a single unified interface.

Node.js ≥ 20 TypeScript Strict MIT License
View on GitHub

Features

⚡

Direct Spawn

Run any AI CLI with a single command. No registration needed. ome spawn --cli claude "Fix the bug"

👥

Employee Registry

Register named employees with preset CLI, model, and role. Dispatch tasks by name.

📄

Job Tracking

Every spawn creates a persistent job with ID, metadata, and streaming NDJSON log. Max 50 jobs, auto-pruned.

👁

Live Observation

Watch running jobs in real-time from any terminal. Cross-process safe file-tailing.

🌐

Web Dashboard

Dark-themed management UI. Employee CRUD, quota config, job monitoring. Zero external dependencies.

🔒

Security Hardened

XSS-safe rendering, path traversal guards, body size limits, localhost-only default bind.

Quick Start

# Install
git clone https://github.com/lidge-jun/ome.git
cd ome && npm install && npm run build

# Seed default employees (Claude, Codex, Gemini)
ome init

# Spawn a one-off agent
ome spawn --cli claude --model opus "Fix the login bug"

# Dispatch to a registered employee
ome dispatch --agent "Claude" --task "Review the PR"

# Watch a job live
ome watch job-m1abc-x9f3

# Start the web dashboard
ome web

Architecture

ome spawn / ome dispatch
  |
  +-- spawn/index.ts --> spawn CLI process
  |     +-- jobs.ts --> ~/.ome/jobs/{id}.meta.json + .ndjson
  |     +-- process-kill.ts --> cross-platform SIGTERM/SIGKILL
  |
  +-- registry/db.ts --> SQLite (employees, sessions, quota)
  |
  +-- observe/
  |     +-- parser.ts --> NDJSON -> ProgressEvent (claude/codex/gemini)
  |     +-- index.ts --> watch() polling + inspect() snapshot
  |
  +-- web/
        +-- routes.ts --> REST API (8 endpoints)
        +-- dashboard.ts --> inline HTML/JS (XSS-safe)

Data Storage

~/.ome/
+-- ome.db              # SQLite: employees, sessions, quota
+-- jobs/               # File-based job tracking
    +-- {id}.meta.json  # Job metadata (atomic write)
    +-- {id}.ndjson     # Streaming event log

Library API

import { spawnAgent, dispatch, initDb } from 'ome';
import { inspect, watch } from 'ome/observe';

initDb('~/.ome/ome.db');

// Direct spawn — jobId available immediately
const { jobId, result } = spawnAgent('Fix the bug', {
    cli: 'claude', model: 'sonnet',
});
console.log(jobId);          // available before completion
const sr = await result;     // wait for output

// Employee dispatch
const output = await dispatch('Frontend', 'Fix CSS grid');
console.log(output.jobId);   // tracking ID

// Observe
const state = inspect(jobId);
for await (const ev of watch(jobId)) {
    console.log(ev);
}
